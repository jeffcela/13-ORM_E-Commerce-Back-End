# Installation

Open terminal/gitbash and run the following:

```
  npm install
```

## Usage

To start please run the following in terminal/gitbash:

```
  mysql -u "your username" -p / enter your password / source db/schema.sql /quit
```
```
  npm run seed
```
```
  npm start
```
